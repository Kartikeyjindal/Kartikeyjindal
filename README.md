## 🚀 Featured Project — Elevate Equity

> A production-grade MERN stack equity crowdfunding platform connecting early-stage Indian startups with retail investors.

**What it does:**
- Investors browse approved startup listings, invest from a digital wallet, track live portfolio P&L, and read aggregated market news from 4 major Indian financial publications
- Founders submit multi-step applications (Basic Info → Financials → Strategy), track fundraise progress, and view their valuation curve via interactive SVG charts
- Admins review/approve startup applications, publish blogs, monitor audit logs, and access platform-wide analytics

**Key engineering highlights:**
- ⚡ **Sub-1s load time** — Vite 8 ESM bundler + Ant Design tree-shaking (~40% smaller bundle vs CRA)
- 🔄 **Zero-downtime DB failover** — HTTP server starts before MongoDB; auto hot-swaps to an in-memory mock store via `require.cache` injection if Atlas is unreachable
- 🗄️ **< 5ms news API** — In-memory Stale-While-Revalidate cache on RSS feeds (5-min TTL, 4 sources)
- 🔐 **Stateless JWT auth** — 3 role types encoded in token payload; no DB session lookup per request
- 💳 **Razorpay integration** — HMAC-SHA256 signature verification server-side before crediting wallet
